# üîß Building
## Dependencies
- [TeXLive, full](https://www.tug.org/texlive/)
- [Make](https://www.gnu.org/software/make/)
- [j2cli](https://github.com/kolypto/j2cli)
## Commands
- **Build the r√©sum√©**:
  ```bash
  make resume
  ```
- **Build the cirriculum vitae**:
  ```bash
  make cv
  ```
- **Build the cover letter**:
  ```bash
  make coverletter
  ```
- **Build all**:
  ```bash
  make all
  ```
- **Purge all**:
  ```bash
  make clean
  ```
- **Open PDFs**:
  ```bash
  make open
  ```
